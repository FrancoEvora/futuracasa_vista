# Futura Casa v8.5 — Mapa Híbrido com Filtros Avançados

Correções e melhorias:
- corrigido o botão de zoom + / - no mapa visual;
- campos de “Explorar lotes por quadra” agora ficam legíveis no mobile;
- filtros adicionados:
  - quadra;
  - disponibilidade;
  - busca textual;
  - área mínima;
  - área máxima;
  - valor mínimo;
  - valor máximo;
- corrigida a lógica para que o filtro “Disponíveis” selecione apenas lotes disponíveis;
- a lista convencional de lotes agora rola dentro de uma subjanela, evitando rolagem longa na página;
- ao selecionar um lote na lista, ele continua sendo marcado no mapa visual.
