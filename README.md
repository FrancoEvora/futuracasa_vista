# Futura Casa v8.2 — IA com salvamento em galeria e simulador financeiro

Ajustes desta versão:
- nos cards de lote, abaixo de **Detalhes**, foi incluído o botão **Gerar imagens por IA**;
- na página/janela de geração de imagens por IA, foi incluído o botão **Salvar imagens na galeria**;
- também foi incluído o atalho **Ver galeria do lote**;
- em cada imagem gerada, o botão **Abrir simulador financeiro** já leva o **lote correspondente** e uma **casa compatível com o estilo selecionado** para a bandeja de simulação financeira.
