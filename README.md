# Futura Casa v12.0 — Modo Guiado

Versão criada para melhorar a compreensão da plataforma pelo cliente.

Principais mudanças:
- Tela inicial com promessa clara:
  "O jeito mais simples de transformar um lote na sua futura casa."
- Três caminhos de entrada:
  - Quero escolher um lote
  - Quero lote + casa
  - Quero simular uma proposta
- Linha do tempo fixa com 5 passos:
  1. Lote
  2. Casa
  3. Lote + casa
  4. Simular
  5. Reservar
- Navegação por timeline e botões no estilo de apps sociais.
- Cards de lote e casa com ação principal mais clara:
  - Escolher este lote
  - Escolher esta casa
- Mapa, assistente e wellness ficam como ferramentas de apoio.
- Textos revisados para falar diretamente com o cliente.
- Mantido formato single-file sem pasta de imagens.
